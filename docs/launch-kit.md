## GitHub description

## Reddit / Hacker News post

Built a small tool called `datareadme`.

Point it at a CSV or TSV file and it generates a clean `DATA_README.md` with:

- a short dataset summary
- inferred column descriptions
- data quality notes
- a pandas loading snippet

The goal is not a giant profiling dashboard. It is a Markdown file you can actually keep next to the data in a repo.

Would love feedback on whether the output feels useful on real-world datasets.
